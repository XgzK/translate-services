# DeepLX 兼容 AI 翻译接口 API 文档

## 接口简介

本项目提供一个与 DeepLX API 完全兼容的翻译服务，后端由强大的大型语言模型（LLM）驱动。它旨在为需要高质量、高兼容性翻译服务的开发者和用户提供一个无缝的替代方案。

此接口特别适合用于沉浸式翻译、Zotero等需要 DeepL 格式的场景，同时享受大模型带来的更佳翻译质量和灵活性。

**接口地址:** `https://deeplx.jayogo.com`

---

## API 文档

接口遵循 DeepLX 的规范，使用简单直观。您可以通过标准的 HTTP POST 请求与 API 进行交互。

### 请求结构

- **Method:** `POST`
- **Endpoint:**
  - 默认模型: `https://deeplx.jayogo.com/translate/{your_api_key}`
  - 指定模型: `https://deeplx.jayogo.com/translate/{your_api_key}/{model_name}`
- **Headers:**
  - `Content-Type: application/json`

### 支持的模型

通过在 Endpoint 中指定 `{model_name}` 参数，可以使用不同的 AI 模型进行翻译。

## API 请求示例

### 请求头 (Headers)

```http
Content-Type: application/json
```

### 路由 (Endpoint)

**使用默认模型：**
```
POST https://deeplx.jayogo.com/translate/{your_api_key}
```

**使用指定模型：**
```
POST https://deeplx.jayogo.com/translate/{your_api_key}/{model_name}
```

示例：
- 默认模型：`https://deeplx.jayogo.com/translate/sk-xxxxxxxxxxxx`
- 指定模型：`https://deeplx.jayogo.com/translate/sk-xxxxxxxxxxxx/gemini-1.5-pro-latest`

### 请求参数 (Request Body)

```json
{
  "text": "Hello, world!",
  "source_lang": "EN",
  "target_lang": "ZH"
}
```

**参数说明：**

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `text` | string | ✅ 是 | 需要翻译的文本，可包含换行符 |
| `source_lang` | string | ❌ 否 | 源语言代码，留空则自动检测（建议填写以提高准确性） |
| `target_lang` | string | ✅ 是 | 目标语言代码 |

### 返回值 (Response)

**成功响应 (200 OK)：**

```json
{
  "alternatives": [],
  "code": 200,
  "data": "你好，世界！",
  "id": 8356681003,
  "method": "Free",
  "source_lang": "EN",
  "target_lang": "ZH"
}
```

**响应字段说明：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `alternatives` | array | 备选翻译结果（通常为空数组） |
| `code` | int | 状态码，200表示成功 |
| `data` | string | 翻译后的文本 |
| `id` | int64 | 请求唯一标识ID |
| `method` | string | 翻译方法标识 |
| `source_lang` | string | 检测到或指定的源语言代码 |
| `target_lang` | string | 目标语言代码 |

## 错误处理

当请求失败时，API 会返回相应的 HTTP 状态码和错误信息。

常见错误码：
- `400` - 请求参数错误
- `401` - API密钥无效或未提供
- `429` - 请求频率超限
- `500` - 服务器内部错误

## 配置说明

### 沉浸式翻译配置要点

1. 将完整接口地址（包含API密钥）粘贴到API URL输入框
2. **必须设置**：每次请求最大段落数设置为 **100**
3. 可选：启用富文本翻译功能

### Zotero 配置要点

1. 在翻译设置中选择 "DeepLX(API)" 服务
2. 在接口配置中填入完整的接口地址（包含API密钥）
3. 保存配置后即可使用

---

## 注意事项

1. **API密钥安全**：请妥善保管您的API密钥（sk-开头），不要在公开场合泄露
2. **语言代码**：建议明确指定 `source_lang` 以获得更准确的翻译结果
3. **文本长度**：单次翻译的文本不宜过长，建议分段处理长文本
4. **请求频率**：请注意控制请求频率，避免触发限流
